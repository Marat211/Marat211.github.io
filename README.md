# Marat211.github.io

This repository is set up as a Jekyll blog for GitHub Pages.

## Getting started

1. Install dependencies:

```bash
bundle install
```

2. Preview locally:

```bash
bundle exec jekyll serve --livereload
```

3. Open `http://127.0.0.1:4000` in your browser.

## Project structure

- `_config.yml` — Jekyll site configuration
- `index.md` — homepage content
- `about.md` — about page
- `_posts/` — blog posts
